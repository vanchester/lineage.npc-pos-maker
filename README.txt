Возможности программы:
1. Генерация кода по выбранным настройкам программы
2. Автоматическое чтение координат и угла поворота камеры из запущенного клиента игры
3. Возможность задания параметров мейкера из списка
4. Сохранение сгенерированного кода в файл или копирование в буфер обмена (как всего кода, так и выделенной части).

Для поиска адреса памяти следует воспользоваться программой ArtMoney. Искать следует значение координаты X среди "С точкой 4 байта". После нескольких отсеиваний необходимо взять самый первый найденный результат в списке.
